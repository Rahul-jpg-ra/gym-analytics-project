# 🏋️ Gym Analytics Project

An end-to-end **Data Analytics Project** built using **Excel, SQL, Python, and Power BI** to analyze gym operations and generate actionable business insights.

This project demonstrates the complete analytics workflow—from raw data preparation to dashboard creation and business intelligence reporting.

---

# 📌 Project Overview

The objective of this project is to analyze gym membership and operational data to answer key business questions related to:

- Member Growth
- Revenue Analysis
- Trainer Performance
- Attendance Analysis
- Customer Retention
- Fitness Goals
- Membership Plans

The project follows a complete analytics pipeline:

**Excel → SQL → Python → Power BI**
