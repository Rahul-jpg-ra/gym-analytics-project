# 📊 Gym Analytics - Excel Dashboard

## 📌 Project Overview

This Excel dashboard provides an interactive analysis of gym membership data using Pivot Tables, Charts, Slicers, and KPI Cards. The dashboard helps understand member distribution, revenue, and performance metrics.

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Formulas

---

## 📂 Dataset

The dataset contains information about:

- Members
- Membership Plans
- Trainers
- Attendance
- Payments
- Workout Sessions
- Nutrition
- Body Measurements
- Feedback

---

## 📈 Business Questions Solved

### Membership Analysis

- Total number of gym members
- Active vs Inactive members
- Members by city
- Members by gender
- Members by membership plan
- Members by fitness goal

### Revenue Analysis

- Total revenue
- Revenue generated by membership plans
- Monthly revenue trend

### Performance Analysis

- Average BMI
- Average retention period
- Members leaving within 90 days
