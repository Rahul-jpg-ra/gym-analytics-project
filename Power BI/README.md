# 📊 Gym Analytics - Power BI Dashboard

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using Gym Analytics data. The dashboard provides key insights into memberships, revenue, member retention, attendance, trainer performance, and fitness trends, helping management make data-driven decisions.

---

## 🛠️ Tools Used

- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations

---

## 📂 Dataset

The dashboard is built using the following datasets:

- Members
- Membership Plans
- Trainers
- Attendance
- Payments
- Workout Sessions
- Nutrition
- Body Measurements
- Feedback

# 📈 Business Questions Solved

## Membership Analysis

- What is the total number of gym members?
- How many members are Active vs Inactive?
- Which city has the highest number of members?
- What is the gender distribution?
- Which membership plan is the most popular?
- What are the most common fitness goals?

## Revenue Analysis

- What is the total revenue generated?
- Which membership plan generates the highest revenue?
- How does monthly revenue change over time?

## Retention Analysis

- What is the average retention period?
- What is the churn rate?
- Which membership plan has the highest retention?
- Which trainer retains members the longest?
- Which city has the highest retained members?

## Attendance Analysis

- Which time slot has the highest attendance?
- Which check-in hour is the busiest?
- How does attendance vary on weekdays and weekends?
- Which day of the week has the highest attendance?
- How has attendance changed over the years?
