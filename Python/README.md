# 🐍 Gym Analytics - Python Data Analysis

## 📌 Project Overview

This project analyzes gym membership data using Python and performs end-to-end exploratory data analysis (EDA) to uncover business insights. The project demonstrates data loading, cleaning, transformation, analysis, and visualization using industry-standard Python libraries.

---

## 🛠️ Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset

The dataset contains the following tables:

- Members
- Membership Plans
- Trainers
- Attendance
- Payments
- Workout Sessions
- Nutrition
- Body Measurements
- Feedback
